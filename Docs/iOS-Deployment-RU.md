# Развертывание приложения MorseLight на физический iPhone

Этот документ содержит пошаговое руководство по сборке и установке приложения **MorseLight** на ваш личный iPhone.

---

## Предварительные требования

Перед началом убедитесь, что у вас подготовлены:
1. **Компьютер Mac** с установленным Xcode 15.0 или новее.
2. **iPhone** с версией iOS 17.0 или новее (для виджетов требуется iOS 18.0).
3. **Кабель** для подключения iPhone к Mac (USB-C или Lightning).
4. **Аккаунт Apple ID** (подходит обычный бесплатный личный аккаунт).

---

## Пошаговая инструкция

### Шаг 1: Генерация проекта Xcode

Проект использует инструмент **XcodeGen** для генерации `.xcodeproj` файла из `project.yml`.

1. Установите XcodeGen через Homebrew (если он еще не установлен):
   ```bash
   brew install xcodegen
   ```
2. Перейдите в корневую директорию проекта и сгенерируйте файл проекта:
   ```bash
   xcodegen generate
   ```
   В корневой папке появится файл `MorseLight.xcodeproj`.

---

### Шаг 2: Открытие проекта в Xcode

Дважды кликните по сгенерированному файлу `MorseLight.xcodeproj` или запустите команду в терминале:
```bash
open MorseLight.xcodeproj
```

---

### Шаг 3: Настройка Apple ID в Xcode

Для подписи приложения вам необходимо добавить свой бесплатный аккаунт разработчика:

1. Откройте настройки Xcode: нажмите `Cmd + ,` (или в меню выберите **Xcode > Settings...**).
2. Перейдите во вкладку **Accounts**.
3. Нажмите кнопку **`+`** в левом нижнем углу и выберите **Apple ID**.
4. Введите данные вашего аккаунта Apple ID.

---

### Шаг 4: Настройка автоматической подписи (Automatic Signing)

Apple требует, чтобы все приложения, запускаемые на физических устройствах, были подписаны сертификатом.

1. В левой панели Xcode (Project Navigator) выберите самый верхний элемент — проект **MorseLight**.
2. В основном окне выберите цель (Target) **MorseLight** во вкладке **Targets**.
3. Перейдите во вкладку **Signing & Capabilities**.
4. Убедитесь, что включена галочка **Automatically manage signing**.
5. В выпадающем списке **Team** выберите ваш добавленный Apple ID (будет помечен как *Personal Team*).
6. **Критически важно:** Измените **Bundle Identifier** (например, с `com.goodmai.MorseLight` на `com.ваше_имя.MorseLight`). Идентификатор должен быть абсолютно уникальным во всем мире, иначе Apple вернет ошибку, что данный Bundle ID уже занят.
7. Повторите пункты 5 и 6 для цели **MorseLightWidget** (указав уникальный Bundle ID, например `com.ваше_имя.MorseLight.widget`).

---

### Шаг 5: Подготовка iPhone (Включение режима разработчика)

Начиная с iOS 16, для запуска неопубликованных приложений на устройстве необходимо включить режим разработчика.

1. Отключите iPhone от Mac (если он подключен).
2. На iPhone перейдите в **Настройки > Конфиденциальность и безопасность** (*Settings > Privacy & Security*).
3. Прокрутите список в самый низ и выберите **Режим разработчика** (*Developer Mode*).
4. Включите тумблер **Режим разработчика**.
5. Телефон попросит перезагрузку. Нажмите **Перезагрузить**.
6. После перезагрузки разблокируйте телефон и во всплывающем окне выберите **Включить** (*Turn On*), затем введите ваш пароль разблокировки устройства.

---

### Шаг 6: Подключение и доверие устройству

1. Подключите iPhone к Mac с помощью кабеля.
2. При первом подключении на экране iPhone появится запрос: **Доверять этому компьютеру?** (*Trust This Computer?*). Нажмите **Доверять** (*Trust*) и введите пароль разблокировки.
3. В Xcode в верхней панели (панель выбора активной схемы и устройства) нажмите на текущее устройство и выберите в списке ваш подключенный **iPhone** вместо симулятора.

---

### Шаг 7: Запуск сборки и развертывания

1. Нажмите комбинацию клавиш `Cmd + R` или кликните по иконке **Play** (треугольник) в левом верхнем углу Xcode.
2. Xcode соберет проект и начнет установку на ваш iPhone.

> [!WARNING]
> При первой установке Xcode может выдать ошибку вида **"Untrusted Developer"** (Ненадежный разработчик). Это нормальное поведение для бесплатных аккаунтов. Приложение установится, но не запустится.

---

### Шаг 8: Доверие сертификату на iPhone

Если вы получили ошибку ненадежного разработчика на iPhone при попытке открыть приложение:

1. На iPhone перейдите в **Настройки > Основные > VPN и управление устройством** (*Settings > General > VPN & Device Management*).
2. В разделе **ПО разработчика** (*Developer App*) вы увидите свой Apple ID. Нажмите на него.
3. Нажмите кнопку **Доверять "[ваш_Apple_ID]"** (*Trust "[your_Apple_ID]"*).
4. Подтвердите выбор в появившемся окне.

---

## Запуск приложения!

Теперь вы можете запустить **MorseLight** прямо с экрана вашего iPhone. 
Приложение будет работать на вашем устройстве бесплатно в течение **7 дней**. По истечении этого срока для продолжения использования достаточно просто пересобрать проект на устройство заново через Xcode (повторно выполнив Шаг 7).
